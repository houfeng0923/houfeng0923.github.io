

## material design lite 


google 基于 类 web component 方式 实现的一套 MD UI 组件。

在页面onload后，通过扫描class name 升级标签组件。对于动态生成内容，需要手动升级： 

```
componentHandler.upgradeElement(button);
```


目前 停止支持。。
